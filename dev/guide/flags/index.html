Input Flags

Base.getindex — Method
getindex(job::DFJob, flag::Symbol)

Searches through the job's inputs for the requested flag. A Dict will be returned with input names as the keys and the flags as values.

source
Base.setindex! — Method
setindex!(job::DFJob, value, flag::Symbol)
